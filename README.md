# AWS EC2 and S3 Web Hosting Project

This project demonstrates how to host a Node.js web application using AWS EC2 and store images in Amazon S3.

## Services Used
- Amazon EC2
- Amazon S3
- Node.js
- Express.js

## Project Steps
1. Launch an EC2 instance using Ubuntu.
2. Install Node.js and npm.
3. Deploy the Node.js application.
4. Create an S3 bucket.
5. Upload images and set them to public access.
6. Display images in the Node.js application.

## Outcome
The web application runs on EC2 and loads images from S3 storage.
